# Principios

- Markdown es el formato canónico para documentación curada.
- Trazabilidad obligatoria: lo curado referencia a su fuente.
- Separación estricta: `sources/raw/` (original) vs `ingested/` (extracción automatizada) vs `curated/` (destilado) vs `deliverables/` (entregables).
- Plantillas mínimas, opinadas y consistentes.
- Decisiones relevantes se registran como ADR.
- VSCode es el centro de trabajo.
