# 01-structure

Esqueleto reutilizable para aplicar el framework a un proyecto.

## Idea
Aquí vive un “esqueleto de proyecto” que se copia/instala en proyectos reales.

## Resultado esperado
Una estructura estable para separar:
- fuentes originales (`sources/raw/`)
- extracción automatizada (`ingested/`)
- documentación curada (`curated/`)
- entregables (`deliverables/`)
