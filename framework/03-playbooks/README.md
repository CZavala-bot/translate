# 03-playbooks

Playbooks de PromptOps: prompts versionados y reutilizables.

## Objetivo
Guiar a una IA (y a humanos) para producir artefactos consistentes:
requisitos, historias, ADRs, actas, planes, etc.

## Regla
Cada playbook define:
- entradas esperadas
- salida esperada (formato)
- checklist de calidad
